# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with React, Node.js, Express, and PostgreSQL
- User authentication with OTP via Twilio SMS
- Menu browsing and search functionality
- Shopping cart with add/remove items
- Order placement and tracking system
- Multiple address management
- Push notifications for order updates
- Admin dashboard with analytics and charts
- Order management for admins
- Menu item management (CRUD operations)
- User management features
- Offers and discounts system
- Inventory tracking
- Payment integration support
- Review and rating system
- Real-time notifications
- Service worker for PWA capabilities
- Responsive design with Tailwind CSS
- Comprehensive documentation (README, CONTRIBUTING, SECURITY)

### Security
- JWT-based authentication
- bcrypt password hashing
- Environment variable configuration for secrets
- SQL injection prevention with Sequelize ORM
- CORS configuration
- Input validation on all endpoints
